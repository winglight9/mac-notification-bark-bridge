# 设置页重构方案

## 目标

- 把设置页从“配置堆料页”改成“日常开关页”。
- 高级规则不在窗口里手填，避免重影、空白和误操作。
- 把每个功能和它对应的输入项放在一起，减少跳读。

## 这版只解决什么

- 轮询、去重、黑屏保活、系统启动、调试模式这些高频项。
- 规则只展示摘要，不在窗口里编辑。
- 窗口更紧凑，默认不用铺满大屏。

## 信息架构

### 1. 扫描与去重

- 轮询间隔（秒）
- 去重窗口（秒）

### 2. 息屏保活

- 是否开启黑色遮罩
- 延迟多少分钟后启动遮罩

### 3. 系统与调试

- 登录 macOS 时自动启动
- 需要时请求辅助功能权限
- 调试模式：只记录，不发送 Bark

### 4. 规则与文件

- 当前规则摘要
- 打开配置文件
- 从磁盘重载

## 交互原则

- 一个功能和它的填写项放在同一个卡片里。
- 数值输入和单位放同一行，例如“2 秒”“300 秒”“8 分钟”。
- 规则区不再出现规则名称、图标 URL、监听应用这类手填框。
- 高级项统一走配置文件，窗口里只保留入口。

## 布局原则

- 默认窗口缩到中等尺寸，避免大面积空白。
- 使用 2 行 2 列卡片布局：
- 第一行：扫描与去重 / 息屏保活
- 第二行：系统与调试 / 规则与文件
- 底部只保留保存状态和主操作。
